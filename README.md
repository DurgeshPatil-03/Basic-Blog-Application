# BlogSphere - Basic Blog Application

## 📖 Overview
BlogSphere is a full-stack blog application built using Node.js, Express.js, MongoDB, and EJS. It allows users to create, read, update, and delete blog posts through a clean and responsive web interface.

## 🚀 Features

- Create a new blog
- View all blogs
- Edit existing blogs
- Delete blogs
- Responsive design
- MongoDB database integration
- Express.js backend
- EJS templating engine

## 🛠 Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

## 📂 Installation

```bash
npm install
```

Start your local MongoDB server so it is available at:

```bash
mongodb://127.0.0.1:27017/blogDB
```

Then run:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
